How I Linked My Pages Together

I linked this page to my main website page using a relative HTML hyperlink. A relative link allows one page in the same GitHub repository to open another page without needing a full file path. I used the <a> tag with href="index.html", which tells the browser to load the homepage from the same folder. This method works on GitHub Pages because both files are stored together in the repository. I also added a button-style link at the bottom of the page so users can easily return to the previous page.
